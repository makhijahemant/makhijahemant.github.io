Swift Print

Lightweight browser-to-printer bridge for local printing

Open Source

Swift Print is a lightweight open-source utility that enables web applications to send print commands to local printers through a desktop bridge. It is designed for receipts, labels, invoices, and ESC/POS printing workflows where browser-based applications need reliable local printer integration.

Key Highlights

  • Lightweight simple browser-to-printer bridge tool
